# Notes on: Data Structures and Algorithms

## Big O

### Objects

- Object.keys - O(N)
- Object.values - O(N)
- Object.entries - O(N)
- hasOwnProperty - O(1)

### Arrays
- Insertion and Removal
  - Push/Pop (at end) - O(1)
  - Shift/Unshift (at start) - O(N)
  - Concat/Slice/Splice - O(N)
- Sort - O(N * log N)
- Searching - O(N)
  - `indexOf`
- Access - O(1)
